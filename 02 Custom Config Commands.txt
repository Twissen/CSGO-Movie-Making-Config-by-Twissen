GREEN / GREENsky / GREENmodels / GREENweapons / GREENeffects / GREENbloods (If you want record to with mirv stream) " You can choose what ever you need ."

-FEATURES:
FRAGon       -  Loads Preset for Frag Recording
SMOOTHon     - Loads Preset for Smooth(Cinematic) Recording
FRAPSfrag    - Loads Preset for Fraps Frag Recording
FRAPSsmooth  - Loads Preset for Fraps Smooth Recording
SPon         -  Enables Preset for Shadowplay Recording
SPoff        -  Disables Shadowplay Recording Preset
STREAMS      - Activates Mirv Streams Commands
CLEAN        - Loads RAW Stream
CLEANworld   - Loads Clean World Stream with Models
CLEANworld2  - Loads Clean World Stream
GREEN        - Loads Green Stream
GREENsky     - Loads Skybox Green Stream
GREENmodels  - Loads Only Models Green Stream
GREENweapons - Loads Only Weapons Green Stream
GREENeffects - Loads Only Effects Green Stream (Blood,shell)
depth        - Loads Depth of Field Stream
mydepth      - Loads Deptf of Field Stream (my settings)
mydepth24    - Loads High Precision Depth Stream
depthworld   - Loads Depth of Field World Screen
WALLon       - Enables Wallhack Stream
WALLoff      - Disables Wallhack Stream
CAMon        - Loads Mirv Camera
HUD          - Loads Hud Stream
fogcolor0    - Sets Fog Color to Default
fogcolor1    - Fog Color Preset 1
fogcolor2    - Fog Color Preset 2
fogcolor3    - Fog Color Preset 3
fogcolor4    - Fog Color Preset 4
fogcolor5    - Fog Color Preset 5
fogcolor6    - Fog Color Preset 6
res1	     - Loads 720p Resolution Preset
res2	     - Loads 1080p Resolution Preset
res3	     - Loads 1440p Resolution Preset
res4	     - Loads 2160p Resolution Preset
setCAMone    - Loads Mirv Camera Preset 1 (you have to set mirv_cam first)
setCAMtwo    - Loads Mirv Camera Preset 2 (you have to set mirv_cam first)
setCAMthree  - Loads Mirv Camera Preset 3 (you have to set mirv_cam first)
setCAMfour   - Loads Mirv Camera Preset 4 (you have to set mirv_cam first)
setCAMfive   - Loads Mirv Camera Preset 5 (you have to set mirv_cam first)
setCAMsix    - Loads Mirv Camera Preset 6 (you have to set mirv_cam first)
setCAMseven  - Loads Mirv Camera Preset 7 (you have to set mirv_cam first)
AGR          - Enables Mirv AGR with the default settings
AGRweapons   - Enables AGR Recording for Weapons
AGRptiles    - Enables/Disables Mirv Agr Recording for Granades
AGRviewmodel - Enables/Disables Mirv Agr Recording for Viewmodel
Skin         - Enables/Disables Skins
[V]			 - Enables/Disables Vgui (Fixes the Reshade Problem)
[SHIFT+F2]   - Demo Player
[INS]        - Pause Demo
[HOME]       - Resume Demo
[+]          - Increase FOV
[-]          - Decrease FOV
[*]          - Resets FOV Value to Config Default
[R]          - Resets All Settings
[F]          - Fog ON/OFF
[C]          - Enables/Disables Crosshair
[X]          - Changes Skybox
[K]          - Enables Mirv Camera
[L]          - Locks Deathmessages for the player you're spectating
[H]          - Highlights Deathmessages for the player you're spectating
[T]          - Changes Timescale 0.2 / 0
[F6]         - Only Deathnotices - Enables/Disables
[F7]         - Hud Enable/Disable
[F8]         - Viewmodel Enable/Disable
[F9]         - Particles (Smoke+) ON/OFF
[F10]        - Change Weapon Side
[UpArrow]    = 500 FPS  
[DownArrow]  = 0 FPS
[LeftArrow]  = 60  FPS  
[RightArrow] = 300 FPS
[Mouse 3]    - Add campath
[Mouse 4]    - Enable Campath
[Mouse 5 / Z]- Clear All Campaths
[G]          - Goes 15 Seconds Back
RESETmirv    - Cleans All Streams
RESETfog     - Resets Fog Settings
RESETcam     - Removes Mirv Camera
RESETall     - Resets All Settings
